# Linux Executable

`scheme-generator` — готовый исполняемый файл для Linux.

## Использование

1. Скачайте файл
2. Дайте права на исполнение:
   ```bash
   chmod +x scheme-generator
   ```
3. Запустите:
   ```bash
   ./scheme-generator <input_file.yaml> [-o <output_file.drawio>]
   ```

### Требования

- Linux kernel 2.6.32 или новее
- libc (glibc) 2.14 или новее
