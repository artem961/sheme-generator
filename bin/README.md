# Готовые исполняемые файлы (Binaries)

Эта папка содержит скомпилированные версии `scheme-generator` для различных платформ.

## Структура

- **`windows/`** — Исполняемый файл для Windows (`scheme-generator.exe`)
- **`macos/`** — Исполняемый файл для macOS (`scheme-generator`)
- **`linux/`** — Исполняемый файл для Linux (`scheme-generator`)

## Как использовать

Скачайте файл для вашей операционной системы и запустите его:

```bash
# Linux / macOS (после chmod +x):
./scheme-generator <input_file.yaml>

# Windows:
scheme-generator.exe <input_file.yaml>
```

Для подробных инструкций см. корневой [README.md](../README.md) проекта.
